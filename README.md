# Airbnb-Melbourne-Pricing prediction 

Kratkorocno izdavanje nekretnina podrazumeva iznajmljjivanje stambenih objekata kao sto su sobe, apartmani ili kuce, koji se izdaju na kraci vremenski period (od jedne noci do par nedelja). Ovakvo trziste ima veliki uticaj na ekonomiju i turizam.
Projekat se bavi konkretno predikcijom cena smestaja u Melburnu koriscenjem ridge i lasso regresije, random forst i XGBoost modela.
